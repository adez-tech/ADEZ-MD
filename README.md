## 🚀 About ADEZ-MD

**ADEZ-MD** is a powerful WhatsApp Multi-Device bot built with Node.js and Baileys.

Designed for speed, stability and easy customization.

Features:

✅ WhatsApp Multi Device  
✅ QR Login System  
✅ Command Handler  
✅ Observer System  
✅ Express Web Server  
✅ Socket.IO Support  
✅ Supabase Ready  
✅ Render Deployment Ready  


---

## ✨ Features

### ⚡ Core

- Fast WhatsApp connection

- Session management
- Multi device support
